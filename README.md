followed by a screen where she picks where she wants to go for her birthday
(Cheddar's, Bobo's, Fort Bliss, or her own pick).

I really hope she likes it — this is probably the hardest I have ever worked
on anything in my life lolz.

## How to see it

It's a single self-contained `index.html` (no build step). To preview locally,
just open `index.html` in any browser.

## Going live (GitHub Pages)

This repo is set up to host for free on GitHub Pages. Two easy options:

**Option A — Deploy from a branch (simplest):**
1. Go to **Settings → Pages**.
2. Under *Build and deployment*, set **Source: Deploy from a branch**.
3. Choose the branch (e.g. `main`) and folder **`/ (root)`**, then **Save**.
4. After a minute it'll be live at:
   `https://dorianacosta.github.io/natalie-s-bday-/`

**Option B — GitHub Actions (auto-deploys on every push to `main`):**
1. Go to **Settings → Pages**.
2. Under *Build and deployment*, set **Source: GitHub Actions**.
3. The included workflow (`.github/workflows/deploy.yml`) handles the rest.
