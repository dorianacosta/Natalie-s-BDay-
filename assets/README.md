# assets

Media used by the birthday site.

- **`us.jpg`** — ✅ included. The photo of you two that shows on the splash
  screen. Swap it anytime by replacing this file (keep the same name).

- **`song.mp3`** — ✅ included. *"Silhouette" by LoopGarden.* It starts when
  she taps **"Open your surprise 🎁"** and can be paused/played with the
  floating 🎵 button. To change the song, replace this file (keep the name
  `song.mp3`).
