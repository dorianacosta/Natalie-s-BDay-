# assets

Media used by the birthday site.

- **`us.jpg`** — ✅ included. The photo of you two that shows on the splash
  screen. Swap it anytime by replacing this file (keep the same name).

- **`song.mp3`** — ⬜ add this yourself to enable the music.
  Drop an MP3 of *"Girls Love Beyoncé" by Drake* here and name it exactly
  `song.mp3`. It starts when she taps **"Open your surprise 🎁"** and can be
  paused/played with the floating 🎵 button. Until the file exists, the site
  just stays quiet (no errors).

  > Tip: use your own copy of the track — this is a personal gift, so don't
  > host a download of it publicly.
